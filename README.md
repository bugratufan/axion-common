# Axion Common

Shared VHDL modules and utilities for Axion FPGA projects.

## Overview

This repository contains reusable VHDL components that are common across multiple Axion projects.

## Usage

This repository is intended to be used as a Git submodule in other Axion projects.

### Adding to your project

```bash
git submodule add https://github.com/bugratufan/axion-common.git axion-common
```

### Updating to latest version

```bash
git submodule update --remote axion-common
```
